------------------
<player value="?">
------------------

.. warning::
	You can only use these placeholders in DYNAMIC animations.
	
name
===========
	Name (IGN) of player.
	
	.. code-block:: html

		<player value="name">

---------

reconnect_server
================
	Name of server player is about to connect to.
	
	.. code-block:: html

		<player value="reconnect_server">
